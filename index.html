AMD R5 1600

AMD Ryzen 5 1600 is a 6-core 12 threaded CPU, capable of handling many different workloads. This CPU at this price point is the best cost to performance processor. Not only will this CPU be able to overclock, but 2600 will also be able to handle light to medium workloads including video editing and 3D animations.

Cooler Master D92

The Cooler Master D92 is one of Cooler Master's dual CPU Fan coolers which provie a push/pull cooling action to the CPU. Not only is this cooler compact, with two fans, and with custom fan adustments, noise can be reduced further.

MSI x370 SLI Plus

This motherboard is perfect for the build if you do not mind the black and red color scheme. The price and the quality is good enough it adds this motherboard to the build. This offers 4 DIMM slots, for future expansions, 3 PCI-E x16, and x1 slots. It also includes 6 SATA ports, more than enough storages for the build, allowing for future expandability, and a plethora of USB 2.0 and 3.1 ports.

Corsair Vengeance LPX 16GB DDR4-3200

This kit of RAM is good enough for video editing as the speed is fast enough. AMD Ryzen particularly perfers faster RAM, and with 3200 it meets the requriements.

Mushkin Reactor 512GB SSD

A SSD for a Main boot drive is key for a fast computer, Not only is 500GB more than enough for the OS, important and most used applications can be stored on this drive

Seagate Barracuda 2TB HDD

A 2TB HDD is a great additon for this build as it provides more than enough additional storage space for most needs, whather for games, video productions, or for storing more improtant files.

MSI Duke GTX 1080 8GB

The GTX 1080 is already a powerhouse of a graphics card, with this model, there is in total 3 fans, allowing great cooling to this GPU.

Cougar Panzer Mid-Tower

Solid case with two panes of temperated and tinted glass sides, and allows up to six 2.5" drives. It is a solid and sturdy build, which also includes a shouded PSU compartment to hide the PSU, and additoal cables.

Seasonic Focus+ 750W 80+ Gold

Seasonic, an already well respeceted PSU brand, and this PSU allows more upgrades in the future, and can power most builds in this budget. Even better, this power supply is fully modualar, allowing the user to only attact the wires which are needed in that build
